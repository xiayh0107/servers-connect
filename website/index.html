Built for human + Agent workflows

One source of truth for intent, context, and evidence.让意图、上下文和执行结果拥有同一个事实源。

01 / ORGANIZE

Find the right host instantly瞬间找到正确主机

Use reusable tags for projects, environments, roles, or clients. Every tag keeps one colour across sidebar, tables, and pickers.按项目、环境、角色或客户建立标签，每个标签在侧栏、表格与选择器中保持同一颜色。

02 / BROWSE

Reference files without a shell无需终端即可引用文件

Browse remote directories read-only and copy precise host:path references that people and AI agents can understand.以只读方式浏览远程目录，并复制人和 AI Agent 都能理解的精确 host:path 引用。

03 / PROTECT

Keep secrets outside the browser让秘密远离浏览器

Passwords and passphrases stay in the operating-system vault. The local UI and agent-safe CLI reference them without exposing values.密码和口令保留在操作系统保险库中，本地 UI 与 Agent 安全 CLI 只引用、不暴露。
