# servers-connect

Monorepo for **SSH Server Manager** — a local-first SSH host and credential
manager for humans and AI agents.

> 🤖 **Using an AI agent?** The skill deploys to every agent on your machine
> in one line — see the
> [agent deployment guide](ssh-server-manager/docs/ai-agents.md).

- Product source, docs, and Agent Skill: [`ssh-server-manager/`](ssh-server-manager/)
- One-line skill install: `./install.sh` (macOS/Linux) or `install.ps1` (Windows) —
  installs deps, links the skill into every detected agent, runs `doctor`
- Product README: [English](ssh-server-manager/README.md) · [简体中文](ssh-server-manager/README.zh-CN.md)
- Website: [`website/`](website/)
